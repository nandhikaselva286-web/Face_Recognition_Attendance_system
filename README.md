# Face Recognition Attendance System
Python-based attendance system that uses face recognition and uniform or logo verification

## Features

*Face recognition using OpenCV
*Uniform or logo detection
*Entry and exit tracking
*Attendance stored using JSON
*Attendance report generated

## Files

*collect_faces.py   : collects face images
*Trainer.py         : trains the face recognition model
*recogniser.py      : recognises and tracks attendance and uniform/logo detection
*attendance.json    : stores attendance records
*report.py          : generates attendance reports

## Technologies Used

* Python
* OpenCV
* JSON
* Haar Cascade Classifier
* LBPH Face Recognizer
